## 启动步骤

### 1. 安装并登录 Gemini CLI
```bash
npm install -g @google/gemini-cli
gemini
# 选择 "Sign in with Google" 并完成登录
```

### 2. 启动桌面应用
```bash
bun run desktop:dev
```
